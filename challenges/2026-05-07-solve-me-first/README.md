Complete the function `solveMeFirst` to compute the sum of two integers.

## Example

Input: a = 2, b = 3
Return: 5

## Function Description

Complete the function with the following parameters:

- `a`: the first value
- `b`: the second value

Returns
- `int`: the sum of a and b

## Constraints

- 1 ≤ a, b ≤ 1000

## Sample Input

a = 2
b = 3

## Sample Output

5

## Explanation

2 + 3 = 5
