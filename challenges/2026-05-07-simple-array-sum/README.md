Given an array of integers, find the sum of the array.

## Example

Input: ar = [1, 2, 3, 4, 10, 11]
Output: 31

## Function Description

Complete the function with the following parameters:

- `ar`: an array of integers

Returns
- `int`: the sum of all elements in the array

## Constraints

- 1 ≤ n ≤ 1000
- 0 ≤ ar[i] ≤ 1000

## Sample Input

ar = [1, 2, 3, 4, 10, 11]

## Sample Output

31

## Explanation

The sum of all the elements in the array: 1 + 2 + 3 + 4 + 10 + 11 = 31
