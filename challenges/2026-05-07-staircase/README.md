Consider a staircase of size n=6:

```
     #
    ##
   ###
  ####
 #####
######
```

Stairs are drawn using `#` symbols and spaces. The last line is not preceded by any spaces. Each staircase has steps numbered from 0 to n-1.

## Function Description

Complete the function with the following parameters:

- `n`: the staircase size

The function should print a staircase as shown above.

## Constraints

- 0 < n ≤ 100

## Sample Input

6

## Sample Output

```
     #
    ##
   ###
  ####
 #####
######
```
