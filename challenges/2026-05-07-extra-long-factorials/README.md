You will be given a number n, and you need to find and print the factorial of n.

For example, 5! = 1 × 2 × 3 × 4 × 5 = 120

For this problem, n can be up to 100, so the factorial can be extremely large. Use BigInteger to handle very large numbers.

## Function Description

Complete the function with the following parameters:

- `n`: an integer

The function should print the factorial of n.

## Constraints

- 1 ≤ n ≤ 100

## Sample Input

25

## Sample Output

15511210043330985984000000

## Explanation

25! = 15511210043330985984000000
