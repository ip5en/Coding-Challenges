In this problem, you will need to add up some very large numbers. Create a function that returns the sum of an array of long integers.

## Example

Input: ar = [1000000001, 1000000002, 1000000003, 1000000004, 1000000005]
Output: 5000000015

## Function Description

Complete the function with the following parameters:

- `ar`: an array of long integers

Returns
- `long`: the sum of all elements in the array

## Constraints

- 1 ≤ n ≤ 10
- 0 ≤ ar[i] ≤ 10^10

## Sample Input

ar = [1000000001, 1000000002, 1000000003, 1000000004, 1000000005]

## Sample Output

5000000015

## Explanation

The sum of the array elements is: 1000000001 + 1000000002 + 1000000003 + 1000000004 + 1000000005 = 5000000015
